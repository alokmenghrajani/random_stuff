unittest.quaxio.com

unittest.quaxio.com is a service owned by Alok Menghrajani. It contains some stuff useful for testing purpose.

The DNS is hosted with namecheap.com and the service itself runs on Amazon's EC2.

(https://another-unittest.quaxio.com/ is the same server, but will trigger an invalid cert warning.)

useful things

  1. unittest.quaxio.com:9000 is filtered 
    iptables -A INPUT -p tcp --dport 9000 -j DROP
  2. unittest.quaxio.com:9001 is closed and triggers a tcp reset 
    iptables -A INPUT -p tcp --dport 9001 -j REJECT --reject-with tcp-reset
  3. unittest.quaxio.com:9002 is closed and triggers an icmp response 
    iptables -A INPUT -p tcp --dport 9001 -j REJECT --reject-with icmp-port-unreachable
  4. https is enabled (startssl.com's free cert)
  5. http://unittest.quaxio.com/status returns a constant string
  6. http://unittest.quaxio.com/ball.png returns a large image (600x600 pixels)
  7. http://unittest.quaxio.com/spacer.gif is a 1x1 pixel image
  8. http://unittest.quaxio.com/tree.png is actually an animated gif
  9. http://unittest.quaxio.com/redirect_ball.pl redirects to ball.png
  10. http://unittest.quaxio.com/redirect_https_ball.pl redirects to ball.png over https
  11. http://unittest.quaxio.com/redirect_spacer.pl redirects to spacer.gif
  12. http://unittest.quaxio.com/redirect_https_spacer.pl redirects to spacer.gif over https
  13. http://unittest.quaxio.com/redirect_inf.pl redirects to itself
  14. http://unittest.quaxio.com/evil_gzip.pl: a large file (1G) compressed to 1M, crashes some stuff
    (some systems only crash if the response is chunked encoded, which is the case here)
